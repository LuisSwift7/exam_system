
## 10. 表：question_feedback（题目反馈）

用途：存储学生对题目的疑问和反馈。

```sql
CREATE TABLE IF NOT EXISTS question_feedback (
  id BIGINT AUTO_INCREMENT PRIMARY KEY COMMENT '主键ID',
  student_id BIGINT NOT NULL COMMENT '学生ID',
  question_id BIGINT NOT NULL COMMENT '题目ID',
  exam_id BIGINT COMMENT '考试ID',
  content TEXT NOT NULL COMMENT '反馈内容',
  reply_content TEXT COMMENT '回复内容',
  status INT DEFAULT 0 COMMENT '状态 0:未处理 1:已处理',
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  reply_time DATETIME COMMENT '回复时间'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='题目反馈表';
```

## 11. 表：question_review（题目讲评）

用途：存储教师发布的题目讲评内容。

```sql
CREATE TABLE IF NOT EXISTS question_review (
  id BIGINT AUTO_INCREMENT PRIMARY KEY COMMENT '主键ID',
  question_id BIGINT NOT NULL COMMENT '题目ID',
  teacher_id BIGINT NOT NULL COMMENT '教师ID',
  title VARCHAR(255) COMMENT '讲评标题',
  content TEXT NOT NULL COMMENT '讲评内容',
  publish_status INT DEFAULT 0 COMMENT '发布状态 0:草稿 1:已发布',
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='题目讲评表';
```
