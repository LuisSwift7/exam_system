# 数据库设计（第一阶段：用户/登录/验证码）

数据库：MySQL 8.0  
约束要求：不使用物理外键（不创建 `FOREIGN KEY` 约束），仅使用逻辑外键字段。  
时间字段约定：`created_time` 使用默认值，但不加 `NOT NULL`。  

## 1. 表：sys_user（用户表）

用途：系统账户，承载教师/学生两类角色（第一阶段用 `role_code` 简化 RBAC，后续可扩展角色/权限表）。

逻辑外键：
- `create_by`：创建人用户ID（关联 `sys_user.id`，不建物理外键）

```sql
CREATE TABLE IF NOT EXISTS sys_user (
  id            BIGINT        PRIMARY KEY AUTO_INCREMENT,
  username      VARCHAR(64)   NOT NULL,
  password_hash VARCHAR(255)  NOT NULL,
  real_name     VARCHAR(64)   NULL,
  role_code     VARCHAR(32)   NOT NULL,
  phone         VARCHAR(32)   NULL,
  email         VARCHAR(128)  NULL,
  status        TINYINT       NULL DEFAULT 1,
  last_login_time TIMESTAMP   NULL DEFAULT NULL,
  created_time  TIMESTAMP     NULL DEFAULT CURRENT_TIMESTAMP,
  create_by     BIGINT        NULL,
  updated_time  TIMESTAMP     NULL DEFAULT NULL ON UPDATE CURRENT_TIMESTAMP,
  update_by     BIGINT        NULL,
  UNIQUE KEY uk_sys_user_username (username),
  KEY idx_sys_user_role_code (role_code),
  KEY idx_sys_user_status (status)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

- `role_code`：`TEACHER` / `STUDENT` / `ADMIN`
- `status`：`1` 启用，`0` 禁用

## 2. 表：sys_captcha（验证码表，可选）

说明：第一阶段验证码可用内存存储；如果需要多实例/重启不丢，可启用该表（不做物理外键）。

```sql
CREATE TABLE IF NOT EXISTS sys_captcha (
  id            BIGINT       PRIMARY KEY AUTO_INCREMENT,
  captcha_id    VARCHAR(64)  NOT NULL,
  answer_hash   VARCHAR(255) NOT NULL,
  expire_time   TIMESTAMP    NULL DEFAULT NULL,
  created_time  TIMESTAMP    NULL DEFAULT CURRENT_TIMESTAMP,
  status        TINYINT      NULL DEFAULT 1,
  UNIQUE KEY uk_sys_captcha_captcha_id (captcha_id),
  KEY idx_sys_captcha_expire_time (expire_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 3. 表：exam（考试表）

用途：定义一场考试的基本信息。

逻辑外键：
- `create_by`：创建人用户ID（教师ID）

```sql
CREATE TABLE IF NOT EXISTS exam (
  id            BIGINT        PRIMARY KEY AUTO_INCREMENT,
  title         VARCHAR(128)  NOT NULL,
  description   TEXT          NULL,
  start_time    TIMESTAMP     NULL,
  end_time      TIMESTAMP     NULL,
  duration      INT           NOT NULL COMMENT '考试时长(分钟)',
  status        TINYINT       NOT NULL DEFAULT 0 COMMENT '0:未发布, 1:已发布, 2:已结束',
  created_time  TIMESTAMP     NULL DEFAULT CURRENT_TIMESTAMP,
  create_by     BIGINT        NULL,
  updated_time  TIMESTAMP     NULL DEFAULT NULL ON UPDATE CURRENT_TIMESTAMP,
  update_by     BIGINT        NULL,
  KEY idx_exam_status (status),
  KEY idx_exam_time (start_time, end_time)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 4. 表：question（题目表）

用途：存储题库题目。

```sql
CREATE TABLE IF NOT EXISTS question (
  id            BIGINT        PRIMARY KEY AUTO_INCREMENT,
  content       TEXT          NOT NULL COMMENT '题目内容',
  type          TINYINT       NOT NULL DEFAULT 1 COMMENT '1:单选, 2:多选',
  options       JSON          NOT NULL COMMENT '选项列表，如 ["A.选项1", "B.选项2"]',
  answer        VARCHAR(64)   NOT NULL COMMENT '正确答案，如 "A" 或 "AB"',
  analysis      TEXT          NULL COMMENT '解析',
  created_time  TIMESTAMP     NULL DEFAULT CURRENT_TIMESTAMP,
  create_by     BIGINT        NULL,
  updated_time  TIMESTAMP     NULL DEFAULT NULL ON UPDATE CURRENT_TIMESTAMP,
  update_by     BIGINT        NULL
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 5. 表：exam_question_relation（考试-题目关联表）

用途：定义试卷包含哪些题目。

```sql
CREATE TABLE IF NOT EXISTS exam_question_relation (
  id            BIGINT        PRIMARY KEY AUTO_INCREMENT,
  exam_id       BIGINT        NOT NULL,
  question_id   BIGINT        NOT NULL,
  score         INT           NOT NULL DEFAULT 1 COMMENT '分值',
  sort_order    INT           NOT NULL DEFAULT 0,
  UNIQUE KEY uk_exam_question (exam_id, question_id),
  KEY idx_exam_id (exam_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 6. 表：exam_record（考试记录表）

用途：学生参加考试的记录。

```sql
CREATE TABLE IF NOT EXISTS exam_record (
  id            BIGINT        PRIMARY KEY AUTO_INCREMENT,
  exam_id       BIGINT        NOT NULL,
  student_id    BIGINT        NOT NULL,
  status        TINYINT       NOT NULL DEFAULT 0 COMMENT '0:进行中, 1:已提交',
  score         INT           NULL COMMENT '总分',
  start_time    TIMESTAMP     NULL DEFAULT CURRENT_TIMESTAMP,
  submit_time   TIMESTAMP     NULL,
  created_time  TIMESTAMP     NULL DEFAULT CURRENT_TIMESTAMP,
  KEY idx_exam_student (exam_id, student_id),
  KEY idx_student_exam (student_id, exam_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 7. 表：exam_answer（答题记录表）

用途：学生每一道题的作答情况。

```sql
CREATE TABLE IF NOT EXISTS exam_answer (
  id            BIGINT        PRIMARY KEY AUTO_INCREMENT,
  record_id     BIGINT        NOT NULL,
  question_id   BIGINT        NOT NULL,
  student_answer VARCHAR(64)  NULL COMMENT '学生答案',
  is_correct    TINYINT       NULL COMMENT '是否正确',
  is_marked     TINYINT       NOT NULL DEFAULT 0 COMMENT '是否标记/草稿',
  created_time  TIMESTAMP     NULL DEFAULT CURRENT_TIMESTAMP,
  KEY idx_record_question (record_id, question_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 8. 表：feedback（题目反馈表）

用途：学生对题目有疑问时提交的反馈。

```sql
CREATE TABLE IF NOT EXISTS feedback (
  id            BIGINT        PRIMARY KEY AUTO_INCREMENT,
  question_id   BIGINT        NOT NULL,
  student_id    BIGINT        NOT NULL,
  content       TEXT          NOT NULL COMMENT '疑问内容',
  reply         TEXT          NULL COMMENT '教师回复',
  status        TINYINT       NOT NULL DEFAULT 0 COMMENT '0:未回复, 1:已回复',
  created_time  TIMESTAMP     NULL DEFAULT CURRENT_TIMESTAMP,
  reply_time    TIMESTAMP     NULL,
  KEY idx_question_student (question_id, student_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```
