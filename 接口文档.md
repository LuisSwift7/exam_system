# 接口文档（第一阶段：登录 / 验证码 / 会话）

统一前缀：`/api`

统一响应结构：

```json
{
  "code": 0,
  "message": "OK",
  "data": {}
}
```

### 学生端补充接口说明

**GET** `/api/student/exam-taking/{recordId}/result`

新增返回字段：
- `ranking.classId`：班级ID
- `ranking.className`：班级名称
- `ranking.myRank`：当前学生班级排名
- `ranking.participantCount`：已交卷人数
- `ranking.classStudentCount`：班级总人数
- `ranking.leaderboard`：班级排行榜列表

**PUT** `/api/student/wrong-book/{id}/note`

用途：保存或更新错题笔记。

请求体：
```json
{
  "note": "这题要先看同比和环比口径"
}
```

**GET** `/api/notifications/{id}`

用途：查看当前登录学生收到的单条通知详情。

### 智能组卷补充说明

**POST** `/api/teacher/exams/genetic-auto-compose`

说明：
- `questionCount` 表示最终试卷总题数
- `typeConfigs[].count` 表示每种题型的最低数量，系统会先满足最低数量，再补足到总题数
- `knowledgePointRequirements` 表示知识点最低覆盖数量，不作为候选池的硬过滤条件
- 如果题库无法满足总题数或最低题型数量，接口会直接返回错误，不会生成少题试卷

- `code=0` 表示成功
- `code!=0` 表示失败，`message` 为可读错误信息

## 1. 获取验证码

**GET** `/api/auth/captcha`

用途：登录前获取验证码图片与验证码ID。

响应 `data`：

```json
{
  "captchaId": "cpt_9f8c2c2b7f7a4a8c8b2d3d1d2e3f4a5b",
  "imageBase64": "data:image/png;base64,iVBORw0KGgoAAA..."
}
```

错误码：
- `1001`：验证码服务不可用

## 2. 登录

**POST** `/api/auth/login`

请求体：

```json
{
  "username": "20231234",
  "password": "123456",
  "captchaId": "cpt_9f8c2c2b7f7a4a8c8b2d3d1d2e3f4a5b",
  "captchaCode": "a7k3"
}
```

响应 `data`：

```json
{
  "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9....",
  "tokenType": "Bearer",
  "expiresIn": 7200,
  "user": {
    "id": 1,
    "username": "20231234",
    "realName": "张三",
    "roleCode": "TEACHER"
  }
}
```

错误码：
- `2001`：用户名或密码错误
- `2002`：验证码错误或过期
- `2003`：账号被禁用

## 3. 注册（学生）

**POST** `/api/auth/register`

用途：学生首次使用注册账号（教师账号由管理员创建）。

请求体：

```json
{
  "username": "20231234",
  "realName": "张三",
  "password": "123456"
}
```

响应：`data=null`

错误码：
- `2004`：账号已存在

## 4. 获取当前登录用户

**GET** `/api/me`

请求头：
- `Authorization: Bearer <accessToken>`

响应 `data`：

```json
{
  "id": 1,
  "username": "20231234",
  "realName": "张三",
  "roleCode": "TEACHER"
}
```

错误码：
- `3001`：未登录或令牌无效

## 5. 考试管理（学生端）

### 5.1 获取可参加的考试列表

**GET** `/api/student/exams`

用途：学生首页展示可参加的考试。

请求参数：
- `page`: 页码，默认1
- `size`: 每页大小，默认10

响应 `data`：

```json
{
  "total": 12,
  "list": [
    {
      "id": 101,
      "title": "2024年春季Java期末考试",
      "description": "请在规定时间内完成所有题目...",
      "startTime": "2024-06-20 09:00:00",
      "endTime": "2024-06-20 11:00:00",
      "duration": 120,
      "status": 1
    }
  ]
}
```

- `status`: `1` 进行中（时间范围内）, `2` 未开始, `3` 已结束

## 6. 考试管理（教师端）

### 6.1 发布试卷

**POST** `/api/teacher/exams/{id}/publish`

用途：教师发布试卷，发布后学生可见。

响应：`data=null`

### 6.2 试卷预览

**GET** `/api/teacher/exams/{id}/preview`

用途：教师预览试卷完整内容，包括题目、分值、答案、解析。

说明：
- 试卷默认总分为 `100` 分
- 组卷时系统会按题量自动分配每题分值
- 题目选项返回为结构化对象，前端应按 `key/value/imageUrl` 渲染

响应 `data`：

```json
{
  "exam": {
    "id": 1,
    "title": "Java期末考试",
    "description": "...",
    "startTime": "2024-06-20 09:00:00",
    "endTime": "2024-06-20 11:00:00",
    "duration": 120,
    "status": 1
  },
  "questions": [
    {
      "id": 101,
      "content": "Java中int占用几个字节？",
      "type": 1,
      "options": [
        { "key": "A", "value": "2", "imageUrl": "" },
        { "key": "B", "value": "4", "imageUrl": "" },
        { "key": "C", "value": "8", "imageUrl": "" },
        { "key": "D", "value": "16", "imageUrl": "" }
      ],
      "answer": "B",
      "analysis": "int占用4字节",
      "difficulty": 1,
      "score": 5
    }
  ]
}
```

## 7. 题目反馈与讲评

### 7.1 学生查看我的反馈

**GET** `/api/student/feedback/list`

用途：查看当前学生提交过的题目反馈。

说明：
- `records[].question` 会返回关联题目
- 学生端仅返回题干、题型、分类、选项等安全字段
- 不返回答案和解析

响应 `data.records[]` 示例：
```json
{
  "id": 1,
  "questionId": 119,
  "examId": 29,
  "content": "这题为什么不能选 A？",
  "replyContent": "先看条件范围。",
  "status": 1,
  "createTime": "2026-03-20T10:00:00",
  "replyTime": "2026-03-20T11:00:00",
  "question": {
    "id": 119,
    "content": "2024 年是农历甲辰年，对应的生肖是：",
    "contentImageUrl": "",
    "type": 1,
    "options": [
      { "key": "A", "value": "兔", "imageUrl": "" },
      { "key": "B", "value": "龙", "imageUrl": "" }
    ],
    "difficulty": 3,
    "category": "常识判断",
    "stemId": null
  }
}
```

### 7.2 教师查看反馈列表

**GET** `/api/teacher/feedback/list`

用途：教师查看学生题目反馈并回复。

说明：
- `records[].question` 会返回完整题目
- 教师端可直接看到题干、选项、答案、解析

响应 `data.records[]` 新增字段：
```json
{
  "question": {
    "id": 119,
    "content": "2024 年是农历甲辰年，对应的生肖是：",
    "contentImageUrl": "",
    "type": 1,

      { "key": "A", "value": "兔", "imageUrl": "" },
      { "key": "B", "value": "龙", "imageUrl": "" }
    ],
    "answer": "B",
    "analysis": "甲辰年对应的生肖是龙。",
    "difficulty": 3,
    "category": "常识判断",

### 7.3 查看题目讲评

**GET** `/api/student/feedback/review/{questionId}`

**GET** `/api/teacher/feedback/review/{questionId}`

说明：
- 返回的讲评对象新增 `question` 字段
- 可用于讲评界面直接展示关联题目

### 7.4 反馈管理中编辑关联题目

**PUT** `/api/teacher/questions/{id}`

用途：
- 教师在反馈管理页中直接修改关联题目
- 可修改题干、题干图片、选项、答案、解析、难度、题库分类
- 资料分析题可同步调整 `stemId`

说明：
- 反馈管理页复用题库管理的题目更新接口
- 保存成功后，当前反馈列表中的关联题目内容应同步刷新

## 8. 考试讲评管理

### 8.1 教师获取讲评列表

**GET** `/api/teacher/reviews`

说明：
- 返回 `examTitle`
- 返回 `summary`
- 返回 `questionReviews`

`questionReviews[]` 字段：
```json
{
  "questionId": 119,
  "questionNo": 3,
  "questionContent": "2024 年是农历甲辰年，对应的生肖是：",
  "content": "这题要先抓住甲辰对应地支。"
}
```

### 8.2 教师创建或编辑讲评

**POST** `/api/teacher/reviews`

**PUT** `/api/teacher/reviews/{id}`

请求体示例：
```json
{
  "examId": 29,
  "title": "三月月考讲评",
  "summary": "这次主要失分在判断推理和数量关系。",
  "questionReviews": [
    {
      "questionId": 119,
      "content": "先看年份干支，再排生肖。"
    },
    {
      "questionId": 121,
      "content": "这题错在把必要条件看成充分条件。"
    }
  ]
}
```

说明：
- 一条讲评可包含一段总评
- 也可包含同一试卷中的多道题目讲评
- `questionId` 必须属于当前 `examId`

### 8.3 教师删除讲评

**DELETE** `/api/teacher/reviews/{id}`

用途：删除一条考试讲评。

### 8.4 学生获取我的考试讲评列表

**GET** `/api/student/reviews`

用途：
- 学生在“我的讲评”页查看自己可见的已发布讲评
- 仅返回当前学生所在班级下、且教师已发布的考试讲评

返回 `data[]` 示例：
```json
[
  {
    "id": 3,
    "examId": 29,
    "examTitle": "三月阶段测验",
    "title": "三月阶段测验讲评",
    "status": 1,
    "summary": "这次主要失分在判断推理和数量关系。",
    "questionReviews": [
      {
        "questionId": 119,
        "questionNo": 3,
        "questionContent": "2024 年是农历甲辰年，对应的生肖是？",
        "content": "先看年份干支，再对应生肖。"
      }
    ],
    "createdAt": "2026-03-20T19:00:00"
  }
]
```

说明：
- 学生端列表页可直接展示 `examTitle`
- 支持显示总评 `summary`
- 支持显示逐题讲评 `questionReviews`

### 8.5 学生按试卷查看考试讲评

**GET** `/api/student/reviews/exam/{examId}`

用途：
- 学生在考试详情页、考试结果页查看该试卷对应的已发布讲评

说明：
- 只会返回当前学生有权限查看的讲评
- 若该试卷暂无讲评，或该学生不属于该试卷所属班级，返回 `data = null`
